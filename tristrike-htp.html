How to Play

  1. This is a two-player game. One player uses X and the other uses O.
  2. The game always starts with X. O plays second.
  3. The board is a 5x5 grid (25 boxes).
  4. Players take turns clicking any empty box to place their symbol.
  5. The first player to make 4 same symbols in a straight line (horizontal, vertical, or diagonal) wins the game.
  6. Only the latest 22 moves stay on the board at any time.
  7. When there are more than 21 moves, the oldest move turns red to show which block will be removed next.
  8. When a new move is made and the limit is exceeded, the red (oldest) block is automatically removed from the board.
  9. You can click the Reset Game button anytime to start a new match.
